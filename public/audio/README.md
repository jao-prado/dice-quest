# Audio Files

Coloque seus arquivos de áudio nas pastas abaixo:

## BGM (músicas de fundo)
```
public/audio/bgm/overworld.mp3   ← menu, item stage, level up
public/audio/bgm/battle.mp3      ← combate normal
public/audio/bgm/boss.mp3        ← combate boss
```

## SFX (efeitos sonoros)
```
public/audio/sfx/click.mp3
public/audio/sfx/attack.mp3
public/audio/sfx/defend.mp3
public/audio/sfx/dice_roll.mp3
public/audio/sfx/hit.mp3
public/audio/sfx/chest_open.mp3
public/audio/sfx/item_collect.mp3
public/audio/sfx/levelup.mp3
public/audio/sfx/gameover.mp3
```

Formatos aceitos: `.mp3` ou `.wav` (renomeie para `.mp3` ou ajuste os caminhos em `src/audio/AudioManager.js`).
